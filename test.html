Bantuan WEB IDE

Fitur Utama:

  • Multi-Proyek - Buat dan kelola banyak proyek
  • Mobile Friendly - Optimasi untuk ponsel
  • Live Preview - Lihat hasil kode secara realtime
  • Syntax Highlighting - Warna untuk HTML, CSS, JS
  • Export - Download proyek dalam berbagai format

Shortcut Keyboard:

  • Ctrl/Cmd + F - Cari teks
  • Tab - Indentasi kode
  • Shift + Tab - Outdent kode

• Gunakan mode landscape untuk pengalaman terbaik di ponsel
